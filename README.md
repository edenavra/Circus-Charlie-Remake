Movement Controls

Arrow Left / A: Moves Charlie to the left

Arrow Right / D: Moves Charlie to the right

Spacebar / W / Arrow Up: Makes Charlie jump

Shield Mechanics

Left Shift + 4: Activates the shield

Left Shift + 5: Deactivates the shield

Game Control Shortcuts

Left Shift + 1: Moves Charlie to the podium to trigger a win condition

Left Shift + 2: Starts the game

Left Shift + 3: Pauses the game

Left Shift + 6: Ends the stage and returns to the main menu

Left Shift + 7: Spawns coins from all active Flaming Pots

Left Shift + 8: Resets the game
